# Cyber-Watch
AI-powered Intrusion Detection and Threat Intelligence Platform

## Features
- Real-time network traffic analysis
- Machine learning based attack detection
- Threat intelligence dashboard
- Website security scanner
- Attack visualization and monitoring

## Technologies
- Python
- Machine Learning
- Flask / FastAPI
- JavaScript Dashboard
- Network security datasets (NSL-KDD, UNSW-NB15, CICIDS2017)

## Project Structure
backend/
frontend/
models/
dataset/

## How to Run
1. Install dependencies
pip install -r requirements.txt

2. Run backend
python app.py

3. Open dashboard
http://localhost:5000
